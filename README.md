# Hesed Claude Plugins

Version: 1.4.0 <!-- x-release-please-version -->

A collection of custom plugins (skills) for [Claude Code](https://claude.com/claude-code) that extend Claude's capabilities with external integrations.

## Installation

Plugins can be installed directly from this marketplace via Claude Code's plugin system.

To install, run /plugin install {plugin-name}@hesed-plugins

or browse for the plugin in /plugin > Discover

or run the following command in your terminal:

```bash
claude plugin marketplace add hesedcasa/claude-plugins
claude plugin install {plugin-name}@hesed-plugins
```
